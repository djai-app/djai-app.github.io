What is this library?

The DJAI Code Generator is a Java and Kotlin compatible library that provides code generation for OpenAPI and AsyncAPI documents. Use this library if you need to generate code for either OpenAPI or AsyncAPI specifications in your own projects.

Key Features

Code generation
Generate code in multiple languages from OpenAPI and AsyncAPI specifications.
Supports both Java and Kotlins
Use this library in your Java or Kotlin based applications! It works the same no matter how you use it.
Convert OpenAPI 2.0 to OpenAPI 3.0
Includes a built-in transformer for easily converting older "Swagger" documents into the newer OpenAPI 3 format.
Supports Mustache Templates
Use Mustache templates to customize the generated code to your needs. The library comes with a set of default templates, but you can also create your own or override existing.
